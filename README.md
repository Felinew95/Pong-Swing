# Pong-Swing

## Description
Pong-Swing est une version simple du jeu Pong développée en Java avec Swing.  
Le projet sert d’entraînement à la programmation orientée objet, à la gestion du clavier et au rendu graphique.

## Fonctionnalités
- Deux joueurs
- Déplacement des raquettes au clavier
- Balle avec rebonds
- Gestion du score et condition de victoire
- Interface graphique avec Swing
- 11 points pour gagner 

## Commandes
| Joueur | Gauche | Droite |
|------|--------|--------|
| Joueur 1 | Q | D |
| Joueur 2 | K | M |

## Technologies
- Java
- Swing (AWT / javax.swing)
